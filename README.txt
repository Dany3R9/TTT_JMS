This is a simple Tic Tac Toe Game developed using Java JMS.
I reccomend to use Joram (http://joram.ow2.org/) as MOM, JORAM incorporates a 100% pure Java implementation of JMS (Java Message Service API released by Sun Microsystem, Inc.). It provides access to a MOM (Message Oriented Middleware), built on top of the ScalAgent D.T. agents based distributed platform.
